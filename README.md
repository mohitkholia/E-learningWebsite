# E-Learning Website

This repository contains a responsive E-Learning Website designed to provide an interactive online education experience.

## 🚀 Live Demo

Check out the deployed site here: [E-Learning Website Live](https://mohitkholia.github.io/E-learningWebsite/)

## Features

- **Responsive Design:** Optimized for desktops, tablets, and mobile devices.
- **Interactive Content:** Engaging educational materials and resources.
- **User-friendly Interface:** Easy navigation and clean layout.
- **Modern Web Technologies:** Built using HTML, CSS, and JavaScript.
- **Customizable:** Easily adapt the content and design to fit various educational needs.

## Technologies Used

- **HTML** (77.4%)
- **CSS** (19%)
- **JavaScript** (3.6%)

## Getting Started

To run the website locally:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/mohitkholia/E-learningWebsite.git
   cd E-learningWebsite
   ```

2. **Open `index.html` in your web browser:**
   - You can simply double-click the file or use a local server for better performance.

## Folder Structure

```
E-learningWebsite/
├── index.html
├── css/
├── js/
├── assets/
└── README.md
```

- `index.html` – Home page of the website
- `css/` – Stylesheets for the website
- `js/` – JavaScript files for interactive functionality
- `assets/` – Images and other media resources
